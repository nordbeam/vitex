Phoenix ❤️ Vite

This Vite dev server isn't meant to be accessed directly.

Visit your Phoenix application at its usual URL, with the Vite dev server running in the background.

Starting Your Phoenix App

Make sure both your Phoenix server and Vite dev server are running:

# Terminal 1 - Start Phoenix
mix phx.server
# Terminal 2 - Start Vite
npm run dev

Then visit http://localhost:4000
